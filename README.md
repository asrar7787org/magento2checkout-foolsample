## Misterge_FoolSample

Magento 2 Checkout customization sample Module

Description
-----------
This module shows some basic customization for magento 2 checkout

Compatibility
-------------
- Magento >= 2.0

This library aims to support and is tested against the following PHP
implementations:

* PHP 7.0
